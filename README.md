# Python 50 Questions

Мой первый проект на Python. Я изучаю программирование самостоятельно с нуля и создал это приложение при помощи искусственного интеллекта Gemini. Программа задает пользователю 50 вопросов, выдает на каждый ответ свои комментарии, проверяет правильность ввода и сохраняет результаты.

## 📌 Основной функционал
- **Интерфейс:** удобное графическое окно с фоновой картинкой (`bg.png`).
- **Валидация данных:** встроенная проверка для имени и возраста.
- **Интерактивность:** живая анкета с индивидуальными комментариями на каждый ответ.
- **Экспорт результатов:** автоматическое сохранение в Excel-таблицу и текстовый отчет.

## 🛠️ Технологии
- Python
- Tkinter
- Pandas & XlsxWriter
- Pillow (PIL)
